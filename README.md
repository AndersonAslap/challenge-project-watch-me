> Sobre o Watchme

<p>
O Watchme é um projeto visando uma plataforma de acesso e exibição de filmes
no menu ao lado o usuário seleciona o gênero do filme e ao lado aparecem os filmes
que pertencem ao gênero escolhido.
</p>
